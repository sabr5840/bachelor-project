Titel:
Hvordan ser jeg min ATP pension

Kort forklaring i naturligt sprog.
Du kan logge ind og se din ATP pension, herunder hvor meget du har indbetalt og hvad du kan forvente at få udbetalt.

Vigtige punkter:
- Se din ATP pension online (ATP, se pension, pensionsoversigt)
- Log ind med MitID
- Se dine indbetalinger
- Se forventet udbetaling
- Opdater oplysninger hvis nødvendigt

Handling (trin-for-trin):
1. Gå til borger.dk
2. Log ind med MitID
3. Åbn din pensionsoversigt
4. Se din ATP pension

Links (start her):
- https://www.borger.dk/Handlingsside?selfserviceId=2eb187ab-d53c-4b88-8698-520b9f99e1e9

Evt. særlige forhold:
Du kan også se ATP via pensionsinfo.dk.

Forbehold:
- Oplysninger afhænger af dine indbetalinger
